Fieldwire

Conditional Input-Fields for Processwire Frontend without jQuery UI

Fieldwire on GitHib

Fieldwire is a replacement for Processwires Inputfield Dependencies. It supports all comparison-operators used by Processwire (=, !=, >, <, >=, <=, %= and *=). Adding custom operators to Fieldwire couldn’t be easier and so I’ve already added the ~= operator (left hand side matches a regular expression).

Fieldwire also introduces field modifiers which transform the value of a field. Modifiers are written using the dot-notation (fieldname.modifiername). Currently Fieldwire offers the lowercase, uppercase and the length modifiers and you can add your own modifiers with just two lines of code.

Examples

age>=18
email*=@gmail.com
textfield!=''
file_input~=^.*\.(jpg|jpeg|png|gif)$
username.length>15
